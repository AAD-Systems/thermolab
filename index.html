Como usar

Exemplo: Vapor superaquecido a P = 1.2 MPa e T = 250 °C.

1. Preencha os limites P₁=1.0, P₂=1.5

2. Preencha os limites T₁=200, T₂=300

3. Preencha os 4 valores de canto da tabela

4. Informe P=1.2, T=250 como alvos

5. Clique Calcular


A interpolação bilinear faz:
1. Interpola em T para P₁ → R₁
2. Interpola em T para P₂ → R₂
3. Interpola em P entre R₁ e R₂

Documentação Técnica

Bem-vindo à documentação do ThermoLab. Aqui você encontra as bases teóricas e guias práticos para utilizar os módulos de interpolação.

1. Interpolação Linear

Dados dois pontos (X₁, Y₁) e (X₂, Y₂), o valor interpolado em Xₜ é obtido por:

Y = Y₁ + (Xₜ - X₁) * (Y₂ - Y₁) / (X₂ - X₁)

Aplicação típica: propriedades de líquido sub-resfriado quando a variação com a temperatura é aproximadamente linear.

2. Interpolação Bilinear

Para tabelas bidimensionais (ex.: vapor superaquecido), a interpolação é feita em três passos:

1. R₁ = f(P₁, Tₜ) via interpolação linear em T
2. R₂ = f(P₂, Tₜ) via interpolação linear em T
3. Y = f(Pₜ, Tₜ) via interpolação linear em P entre R₁ e R₂

3. Tabelas Suportadas

Atualmente, você pode inserir manualmente os dados de tabelas de vapor saturado e superaquecido. A futura integração com CoolProp permitirá chamadas diretas às equações de estado.
